# Projeto 6 - Dashboard Exel
ptbr- Um projeto *EXEMPLIFICATIVO* de um dashboard construído dentro do Excel e a respectiva análise de dados.  

en- An *EXAMPLE* project of a dashboard built within Excel and the respective data analysis.  

# ETAPAS
PT-BR

*1. LIMPEZA DE DADOS*
- Adicionado filtros nas colunas foi verificado os dados que estavam totalmente desalinhados (Ex: na coluna accident_severity havia 49 células com a escrita 'Fetal' foram substituídas pela escrita correta 'Fatal').
- Também foram retirados as céululas vazias. 

*2. PROCESSAMENTO DE DADOS*
- Foi criado duas colunas: 'Month'/'Mês' com a FUNÇÃO text para que ela extraia da coluna 'accident date' apenas o nome do mês e depois a coluna Year/ano.

*3. ANÁLISE E VISUALIZÇÃO DE DADOS*
- Foi inserida várias novas pivot's table para a análise de visualização dos dados. 
- Análise da relação dos dados entre si e as porcentagens entre os níveis do acidente.

*4. CONSTRUÇÃO DE DASHBOARD*
- Foram construídos os gráficos utilizando a conexão com as pivot's table e construído um dashboard para visualização dos dados
- Adicionei um painel com filtros dos anos e pela área dos acidentes - Urbana ou Rural.

# STEPS

EN

*1. DATA CLEANING*

- Filters were added to the columns, and the data that were completely misaligned were checked (e.g., in the accident_severity column there were 49 cells with the term "Fetal", which were replaced with the correct term "Fatal").
- Empty cells were also removed.

*2. DATA PROCESSING*

- Two columns were created: "Month" using the TEXT function to extract only the month name from the accident date column, and "Year".

*3. DATA ANALYSIS AND VISUALIZATION*

- Several new pivot tables were inserted for data visualization analysis.
- Analysis of the relationships between the data and the percentage distribution across accident severity levels.

*4. DASHBOARD CONSTRUCTION*

- Charts were built using connections with the pivot tables, and a dashboard was created for data visualization.
- A filter panel was added for years and for the accident area — Urban or Rural.
